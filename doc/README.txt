Compilation (depuis la racine) via le Makefile :
make

Lancement:
./BatailleNavale

Selon la configuration de la partie (nombre de joueurs, taille de la grille et de la flotte), le jeu peut nécessiter une fenêtre assez grande pour permettre l'affichage de tous les éléments à l'écran.
Nous recommendons l'utilisation du mode plein écran.
